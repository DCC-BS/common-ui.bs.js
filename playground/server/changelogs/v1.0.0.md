# Version 1.0.0
We are excited to announce the release of version 1.0.0 of our Common UI module! This release marks a significant milestone in our journey to provide a robust and user-friendly interface for developers.

## New Features
- **Customizable Components**: Users can now customize UI components to better fit their application's design needs.
- **Improved Performance**: Significant optimizations have been made to enhance the performance of the UI components.
- **Enhanced Accessibility**: We have implemented several accessibility features to ensure our UI is usable by everyone.
- **Dark Mode Support**: A new dark mode option has been added for better user experience in low-light environments.
